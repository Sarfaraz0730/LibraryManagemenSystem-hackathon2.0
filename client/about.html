About Library Management System

Welcome to the Library Management System

Hello, I'm Sarfaraz, and this is my hackathon mini project. I've developed a web-based Library Management System tailored for use by a library manager. This application focuses on managing books and library members, along with handling the issuing and returning of books.

Features

  • Member Management: View, add, delete, and edit library members.
  • Book Management: View, add, delete, and edit books in the library.
  • Book Issuing and Returning: Facilitate the process of issuing and returning books.

Requirements

  • Focus on API and Functionality: Prioritize building robust and efficient APIs.
  • Simple UI: Create a simple yet effective user interface for the library manager.
⚠️ Warning: If the server is not running, the app's functionality may not work as expected.
